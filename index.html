Crystal Collector

How is the game played?

The game will start with a random number showing. The object of the game is to get your "total score" to equal the random number provided.

To start the game, simply click on the crystals below. Each one holds a hidden random number. Each time you click on a crystal, the "total score will increase by they hidden number.

If you match the number exactly, YOU WIN! If you go over the number, you lose. Each time you win or lose, the game will reset and you will have to match a new random number, with different crystal values.
